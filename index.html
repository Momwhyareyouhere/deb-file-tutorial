Introduction

Learn how to create a .deb file, a Debian package format used for installing software on Debian-based systems like Ubuntu.

Requirements

  • Debian-based system (e.g., Ubuntu)
  • Basic knowledge of terminal commands
  • Debian packaging tools (e.g., dpkg-deb, dh_make)

Steps to Create a .deb File

sudo apt-get install build-essential devscripts debhelper
Copied!
mkdir myproject
cd myproject
Copied!
mkdir -p myproject/usr/local/bin
mkdir -p myproject/DEBIAN
Copied!
nano myproject/DEBIAN/control
Copied!
Add the following content: 
Package: myproject
Version: 1.0
Section: base
Priority: optional
Architecture: all
Depends: python3
Maintainer: Your Name <your.email@example.com>
Description: A brief description of myproject.
dpkg-deb --build myproject
Copied!
sudo dpkg -i myproject.deb
Copied!

Troubleshooting

If you encounter issues, check the following:

  • Ensure all required fields are correctly filled in the control file.
  • Verify the directory structure and file permissions.
  • Check the output of the `dpkg-deb` command for errors.
